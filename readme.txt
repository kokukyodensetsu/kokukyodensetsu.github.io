# ブラウザウディタ サンプルゲーム説明書 (2024/08/08)

## はじめに
このゲームはウディタのサンプルゲームをブラウザ上に再現したGame.exe環境で試すデモです  
準備中の表示が消えた後に画面をタッチするとゲームデータのダウンロードがはじまり、  
ゲームデータが完了し次第ゲームが開始されます  
(サンプルゲーム全体で 20MB 程度のダウンロードが発生するので通信環境にはご注意下さい)  

## 基本操作
キーボードからの入力の他、画面スワイプによる移動とボタン操作に対応しています  
サンプルゲームの範囲内でのキーボード入力については、下記のように動作します  

[Zキー] [Enter] => 決定キー (選択肢の選択や会話の進行などに利用します)  
[Xキー] => キャンセルキー (メニュー画面呼び出しなどに利用します)  
[Shift] => サブキー (戦闘中に押すと戦闘が早くなります)  
[方向キー] => メニューや選択肢などを選ぶ  
[F8] => 画面情報の切り替え  
[F12] => ゲームのリセット  

その他、基本的にキー入力の操作はそのままGameに渡されます

## 素材と許諾
サンプルゲームに含まれる素材の権利については
オリジナルの公式サンプルゲームに準拠するということで省略させて頂きます  

オリジナルの ウディタ(WOLF RPG エディター)の公式入手先は下記になります  

https://silversecond.com/WolfRPGEditor/

SmokingWOLF様より許諾を得た上で行なっています  

## ライセンス表記
正式な詳細ライセンスは別途LICENSEファイルに記載しています  
ここでは各種ライセンスの概要のみを記述します  
これらのライセンスはData.wolfを差し替えて頒布する際にも必要です

- DXライブラリ関係について

ブラウザウディタはDXライブラリをウェブブラウザ向けにビルドし利用しているため
DXライブラリ自体(開発者:山田 巧様ならびに先駆者の方々)と
DXライブラリが利用しているソフトウェア(libpng等)のライセンス表記が必要です

https://dxlib.xsrv.jp/dxlicense.html
 
またDXライブラリ部分のウェブブラウザ向けビルドは  
「かめのこにょこにょこ」様のDXライブラリ移植コードを参照させて頂いております  

https://github.com/nokotan/DxLibForHTML5

- ブラウザウディタ自体について

Game.exeの基盤コードはSmokingWOLF様より  
オリジナルのGame.exeソースコードを提供頂いてウェブブラウザ向けに移植しています  

こちらのソースコードは非公開ですが  
成果物(woditor.wasm)の複製及び頒布の権利について許諾して頂いている状態です  

そして移植コードの製作者である私(rikka)からも  
利用者に対してwoditor.wasmならびに関連ファイルの複製及び頒布の権利について許諾致します  

ただし、現時点ではwoditor.wasmを特別な許可なしに
単純な再頒布目的で再頒布することはご遠慮ください  
(ようは開発元を騙られたり善意であったとしてもこちらが管理できない配布元になられると困るという事)  
(ゲーム製作者様がゲームを配布する分には何の問題もないです)
